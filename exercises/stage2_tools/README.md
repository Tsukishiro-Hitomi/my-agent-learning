## 先想清楚这个"一来一回"
```
你 ──(带工具说明书的问题)──▶ 模型
                             │  "我要调 get_weather(city=北京)"  (stop_reason=tool_use)
你 ◀──────────────────────────┘
你执行 get_weather → 把结果用 tool_result 回传（带 tool_use_id）──▶ 模型
                                                                   │ "北京今天晴，25°C"
你 ◀────────────────────────────────────────────────────────────────┘
```
关键点：**模型不自己执行工具，只是"请求"；执行是你代码的事**。回传时 `tool_use_id` 必须对得上。

## 通关标准
- 模型确实调用了 `get_weather`
- 传进去的城市是「北京」
- 最终回答用上了工具返回的天气
